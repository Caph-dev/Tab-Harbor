# Triage Labels

这些技能使用五个 canonical triage roles。此文件把这些角色映射到本仓库 issue tracker 中实际使用的 label 字符串。

| Label in mattpocock/skills | Label in our tracker | 含义 |
| -------------------------- | -------------------- | ---- |
| `needs-triage` | `needs-triage` | 需要维护者评估这个 issue |
| `needs-info` | `needs-info` | 正在等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 描述足够完整，AFK agent 可以接手 |
| `ready-for-human` | `ready-for-human` | 需要人类实现 |
| `wontfix` | `wontfix` | 不会处理 |

当某个技能提到一个角色时，例如 “apply the AFK-ready triage label”，使用上表中对应的 label 字符串。

如果以后仓库改用不同的 label 词汇，直接编辑右侧 `Label in our tracker` 列。
